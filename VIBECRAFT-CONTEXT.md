# VIBECRAFT-CONTEXT.md

Ce fichier centralise le contexte business utilisé par toutes les skills MarketingPilot.

Les skills lisent automatiquement ce fichier au runtime pour personnaliser leurs outputs (ton, ICP, produits, lead magnets, frameworks). Une seule modification ici met à jour les 10 skills.

Si vous utilisez ce package pour votre propre business : remplissez les sections ci-dessous avec votre contexte. Les skills s'adapteront automatiquement.

Si ce fichier est absent ou vide : les skills demanderont le contexte produits à l'utilisateur au premier appel.
